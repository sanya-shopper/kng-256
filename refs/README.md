# refs/ — archived bibliography PDFs

Every work cited in `main.tex` was researched for a **legitimately free**
PDF; whatever exists was archived here so the paper can be read
self-contained. Sources used are exclusively: official standards bodies
(NIST, RFC Editor), the IACR proceedings archive, SpringerLink volumes
that Springer serves freely under the IACR–Springer agreement, arXiv,
Project Euclid / AMS open archives, official project sites run by the
authors, and authors' own academic homepages hosting their own papers.
No pirate mirrors were used. Works with no legal free copy (two
copyrighted textbooks) are listed at the bottom and must be obtained via
a library.

Where the archived file is an author preprint/full version rather than
the publisher's typeset copy, that is noted; pagination may differ from
the citation.
